WeightOfEvidence

Description:

The WeightOfEvidence Operator provides an approach to combine information from multiple sources for decision-making. This Operator provides automatic grouping based on statistics. The user also can manually adjust grouping according to work experience. This Operator is frequently followed by a Logistic Regression Operator.

Algorithm:

WOE(Weight Of Evidence) is based on the statistical Weight of Evidence. WOE is a measure of how much an attribute in the data is related to statuses. Suppose there are two statuses and we want to decide which is more consistent with the given data. First, supply a suggested grouping according to the relationship between an attribute and statuses and the statistic, such as Gini Score, Information Values, and WOE. Specific statistics expression as follows:

  • WOE:

    Gi denotes the record number of Good value in ith group. G denotes the record number of Good value in data. Bi denotes the record number of Bad value in ith group. B denotes the record number of Bad value in data.
  • Information Values(IV):

  • Gini Score:

    Ni denotes the record number in ith group. N denotes the record number in data.

Also, Alpine Miner provides the interactive modification grouping interface. Users can adjust the suggested grouping based on their experience. Alpine Miner will recompute the statistics according to the adjustment.

Properties:

Dependent Column: Define the column used as binary classification variables.
goodValue: The value stored in dependent column represents the event to be analyzed. For example, Active versus Inactive
Setting WOE: Click the "Setting WOE" button to open the dialog for viewing the statistics based on the suggested grouping and manually adjusting the grouping.

The top left of the dialog shows the available columns list. The top right of the dialog shows the statistics based on grouping (Gini Score, Information Values). On the bottom of dialog, the user can modify the grouping through clicking the "Add" button or "Delete" button. When the user clicks the "Calculate" button after adjustment, the statistics on top right will be recomputed. When the user clicks the "Auto" button, the suggested grouping of the selected attribute is provided.

Press the "Auto Grouping" button to provide the suggested grouping of all attributes.

Press the "OK" button to commit the selection changes.

Press the "Cancel" button to cancel all the selection changes.
columnNames: Click the "Column names" button to open the dialog for selecting the available columns from the input dataset for analysis.

Check / uncheck the box in front of the column names to select / de-select the column.

Check the "Select All" box to select all the columns.

Press the "OK" button to commit the selection changes.

Press the "Cancel" button to cancel all the selection changes.

Input:

  • A dataset from the preceding operator.

Output:

  • Visual Output: WOE Grouping Result.
  • Output to succeeding operator: WOE Grouping Rules.
